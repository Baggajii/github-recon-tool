GitHub Recon Automation Script
Author: Ramandeep Bagga
1. Overview

This script scans public GitHub repositories to find files that contain a specific keyword or domain. It uses the GitHub Code Search API and collects information like repository name, file path, file type, matched snippet, URL, and timestamp.
All results are saved in a file named results.json.

2. Features

Keyword or domain input through console
GitHub Personal Access Token input
Searches across public GitHub repositories
Extracts file details and a cleaned snippet
Masks long sensitive values inside the snippet
Handles API errors and rate limits
Saves results in structured JSON format

3. How to Run

Install dependencies
pip install -r requirements.txt

Run the script
python3 main.py

Enter the keyword or domain
Enter your GitHub API Token
The script will generate results.json automatically

4. Files in This Project

main.py — Main scanning script
requirements.txt — Required dependency
results.json — Sample output generated by the script
README.txt — Documentation guide

5. Notes

This tool is for cybersecurity training and OSINT purposes
Do not share your GitHub token publicly
A stable internet connection is required
If you hit the rate limit, try after a short break
